# Python Jokes with `pyjokes`

The `pyjokes` library is a fun way to get one-liners for programmers.

### Installation
```bash
pip install pyjokes
```

### Usage
Here is how you can use it in your code:

```python
import pyjokes

# Get a random geeky joke
joke = pyjokes.get_joke()
print(joke)
```

### CLI Support
You can also run it directly from your terminal:
```bash
pyjoke
```
